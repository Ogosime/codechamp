MERN Stack: Definition

  • M - MongoDB (NoSQL Database Management)
  • E - ExpressJS (NodeJS Framework for BackEnd Design and building RESTful APIs)
  • R - ReactJS (JavaScript Library for FrontEnd)
  • N NodeJS (BackEnd JavaScript Runtime Environment)
